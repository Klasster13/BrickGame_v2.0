Макросы

◆ COLOR

#define COLOR   1

Режим использования цветов для фигур

1 - цветной режим, 0 - монохромный режим

◆ DROP

#define DROP    1

Режим для кнопки вниз

1 - нажатие вниз перемещает фигуру сразу максимально вниз, 0 - нажатие вниз двигает фигуру вниз на одну строку, вне зависимости от таймера

◆ FIGURES

#define FIGURES   7

Параметры фигуры

Кол-во фигур в игре

◆ MODE

#define MODE    0

Режим подсчета уровня

1 - новый уровень каждые 600 очков(по заданию), 0 - новый уровень каждые 10 удаленных линий

◆ ROTATION

#define ROTATION   1

Режим вращения фигуры

1 - против часовой, 0 - по часовой
